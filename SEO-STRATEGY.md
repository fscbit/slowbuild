## 四、Schema 结构化数据方案

### 4.1 首页需要的全部 JSON-LD

| Schema 类型 | 用途 | 优先级 |
|-------------|------|:---:|
| **WebSite** | 告诉搜索引擎网站名称、URL、搜索功能 | 🔴 必须 |
| **Organization** | 品牌信息、Logo、社交媒体链接 | 🔴 必须 |
| **ItemList** | 首页工具列表（让 Google 理解工具有哪些） | 🟡 推荐 |
| **BreadcrumbList** | 面包屑导航（提升搜索结果展示） | 🟡 推荐 |
| **BlogPosting** | 博客文章（每篇博客页单独用） | 🟡 推荐 |
| **Product** | Shop 产品（产品页单独用） | 🟢 可选 |
| **FAQPage** | FAQ 页（富文本结果） | 🟢 可选 |

### 4.2 完整 JSON-LD 代码

将以下代码整体放入 `<head>` 内的 `<script type="application/ld+json">` 标签中：

